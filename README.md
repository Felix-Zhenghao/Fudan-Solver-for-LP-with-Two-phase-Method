## Fudan University Linear Programming Solver！

The solver can output the specific steps of solving a linear programming problem and you can check your homework answer with it!

### User's Guide
#### 0 Install the libraries
pip
```
pip install FDSMLP -i https://pypi.org/project
```

#### 1 Import the libraries
```python
from FDSMLP.Simplex import Two_step_simplex
```

#### 2 Input
```python
A = [
    [1, 1, 2], 
    [-1 2, 1],
    [1, 0, -2]
]# Coefficients of constraints
b = [1, 3, -1]# Coefficients of the object function. If len(b)<# col of A, all omitted zeros will be added automatically.
constrain = [4, 4, 5]# Those constant b's.
constraints =["<=","=",">="]#  Types of inequations of constraints
      
# Initialize the solver 
simplex=Two_step_simplex(b,A,constrain,constraints)
```

#### 4 Solve
```python
from FDSMLP.Simplex import Two_step_simplex

A = [
    [0, 1, 2, 1, 0], 
    [-1, 2, 1, 1, 1],
    [3, 0, 3, 1, 0], 
]
b = [1, 3, -1]
constrain = [4, 4, 4]
constraints =["=","=","="]
simplex=Two_step_simplex(b,A,constrain,constraints)
simplex.solve()
```

#### 5 How to use the parse_input function
```python
from FDSMLP.core.utils.input_parser import parse_input
from FDSMLP.Simplex import Two_step_simplex


constrain,A,b,constraints=parse_input()

simplex=Two_step_simplex(b,A,constrain,constraints)
simplex.solve()
```
